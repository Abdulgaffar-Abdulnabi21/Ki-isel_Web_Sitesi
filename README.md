# WEB-PROJE

Bu proje, kişisel tanıtım amacıyla hazırlanmış çok sayfalı bir web sitesi çalışmasıdır.  
Proje içerisinde kullanıcıya ait kişisel bilgiler, özgeçmiş, şehir tanıtımı, kültürel miras içerikleri, ilgi alanları, iletişim formu ve giriş sistemi yer almaktadır.

## Proje Amacı

Bu projenin amacı, HTML, CSS, JavaScript, Vue.js ve PHP kullanarak temel web programlama konularını uygulamalı şekilde göstermektir.

Proje kapsamında

- Çok sayfalı site yapısı kurulmuştur
- Responsive ve düzenli bir arayüz hazırlanmıştır
- İletişim formu doğrulamaları yapılmıştır
- PHP ile form verileri işlenmiştir
- Basit bir login sistemi hazırlanmıştır
- JavaScript ve Vue.js ile istemci tarafı kontroller eklenmiştir

---

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- Vue.js
- PHP
- XAMPP  Apache
- Bootstrap
